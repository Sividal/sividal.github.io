Политика конфиденциальности

Настоящая Политика конфиденциальности действует в отношении всей информации, которую данный сайт может получить о Пользователе во время использования им данного ресурса.

1. Сбор персональных данных

Мы собираем только те данные, которые вы добровольно оставляете в формах обратной связи для получения консультации по продукту «Сивидал»: имя и номер телефона.

2. Цели обработки данных

Ваши данные используются исключительно для связи с вами, предоставления информации о специализированном продукте диетического питания.

3. Использование сторонних форм (iframe)

Для приема заявок на сайте используются программные модули (iframe) сторонних партнерских сервисов. Заполняя эти формы, вы соглашаетесь с тем, что ваши данные передаются непосредственно оператору партнерской программы для обработки заявки.

4. Защита информации

Администрация сайта принимает необходимые технические меры для защиты вашей личной информации от несанкционированного доступа третьих лиц.

5. Файлы Cookie

Сайт использует cookie для анализа посещаемости и корректной работы интерфейса. Вы можете отключить их в настройках вашего браузера.
